如何传递一个文件到任意一台电脑，想到最有效的方式就是通过浏览器进行上传和下载，这就涉及到了上传和下载服务的搭建，当然搭建方法很多，下面给出最简单快速的方式


* 下载服务

```bash
python -m SimpleHTTPServer port
```
port为端口

* 上传服务

```bash
python <( curl http://tmp.qn.wei-ju.com.cn/uploader ) port
```
port为端口
