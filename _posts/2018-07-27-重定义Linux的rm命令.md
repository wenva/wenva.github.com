`rm -rf`可以说是非常致命，删除之后比较难恢复，因此我们尽量减少直接使用，可以参考mac、window都有一个回收站，因此我们可以重定义rm命令

```bash
echo -e 'alias rm=trash\ntrash() {\n mkdir -p ~/.trash;/bin/mv $@ ~/.trash/\n}' >> ~/.bash_profile
```

这样即使我们误删，也可以从~/.trash中找到，但要记得定时清理回收站
